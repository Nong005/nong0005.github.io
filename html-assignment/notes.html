HTML tags and Description

<em>
- Used for string of emphasized, slightly more important text.
<strong>
- Used for string of highly emphasized, much more important text.
<mark>
- Used to highlight a piece of text.
<q>
- Used for a small quotation embedded within a content.
<cite>
- Used for a citation for another source, often used with quotations.
